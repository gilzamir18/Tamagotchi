## Detalhes do projeto

Neste projeto, utilizo a PokeAPI para obter informações sobre pokemons que você pode adotar
como seu mascote virtual.

### Ferramentas
Este projeto está estruturado no Visual Studio 2022 Community.
Acredito que facilmente pode ser adaptado para outra ferramenta,
como o Visual Studio Code. O sistema operacional de desenvolvimento
é o Windows 11, mas não se usa nenhuma recurso específico deste
sistema. Facilmente o projeto pode ser migrado para outro sistema
operacional.
